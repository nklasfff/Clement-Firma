# Clement WellbeingAtWork

Wellness-framework til arbejdspladsen — baseret på Clements terapeutiske cirkelmodel, tilpasset kontormiljøer.

## Struktur

```
├── docs/
│   └── designskitse.md      # Fuld designskitse med 7 cirkler, perspektiver, temaer og øvelser
├── config/
│   └── firma.yaml            # Virksomhedstilpasning (logo, farver, branche)
├── assets/
│   └── images/               # Hero-billede og cirkelfigur (kopieres fra clement-app)
└── exercises/                # Individuelle øvelsesfiler (kommer)
```

## Kom i gang

1. Kopiér `IMG_2068.png` → `assets/images/hero.png`
2. Kopiér `IMG_2066.png` → `assets/images/cirkler.png`
3. Tilpas `config/firma.yaml` til virksomheden
4. Se `docs/designskitse.md` for det fulde framework
